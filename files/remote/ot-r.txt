NGINX is an open source web server and reverse proxy that excels at large-scale web integration, application security, and web acceleration. NGINX Plus extends NGINX with additional load balancing and application delivery features. The articles in the NGINX Plus Admin Guide will quickly show you how to use some of the most popular features of NGINX and NGINX Plus.
